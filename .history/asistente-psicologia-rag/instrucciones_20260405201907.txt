
1. instalar dependencias con UV

    Crear ambiente virtual
    - uv venv


    Activar ambiente virtual
    - .\.venv\Scripts\activate

     instalar dependencias en el ambiente virtual
    - cd asistente-psicologia-rag
    - uv pip install -r .\requirements-nuevo.txt
    - uv pip install requirements.txt


2. Activar el entorno virtual 

3. Crear la bd chromadb, acceder a chatbot/chromadb_utils
  
    - cd chatbot/chroma_db-utils 
    - python create_chroma_db.py
    - python load_chroma_db_data.py

4. ejecutar la aplicación (volver al directorio raiz de la app)
    Asi para ejecutarla en la red local y poder hacer las peticiones (todo esto se hace desde la raiz)
    - uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

    asi para ejecutar en la terminal el chatbot
    - python -m chatbot.main
